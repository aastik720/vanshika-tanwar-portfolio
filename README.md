# Vanshika Tanwar Portfolio

This is a standalone portfolio website for Vanshika Tanwar.

## Files

- `index.html` - main portfolio page
- `style.css` - design, layout, and animations
- `script.js` - scroll reveal animation
- `assets/vanshika-avatar.svg` - local profile illustration

## Upload To GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to repository `Settings`.
4. Open `Pages`.
5. Select branch `main` and folder `/root`.
6. Save and wait for GitHub to publish the website.
